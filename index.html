The lab

Six steps. Six prompts. Copy, paste, go.

Do them in order — each builds on the last, so keep the same Claude chat open. Hit Copy, paste into Claude, and (for steps 1–2) attach the CSV you downloaded above.

First time?

Two easy ways to do this.

You just need a Claude account. Pick whichever feels comfortable — the prompts are identical.

Easiest

Claude.ai (in your browser)

Go to claude.ai, start a new chat, click the 📎 paperclip to upload nice_cxone_churn.csv, then paste the Step 1 prompt. That's it.

For terminal folks

Claude Code (desktop / CLI)

Unzip the lab into a folder, open Claude Code there, and just mention the file by name — Claude can read everything in the folder. No uploading needed.
